Keibamon 競馬モン

A local-first data & machine-learning platform for Japanese horse racing (JRA / NAR) — built to turn open racing data into honest, testable insight.

ALPHA · ANALYZER UI IN PROGRESS

// Objectives

DATA PLATFORM

Medallion data lake

JRA-VAN fundamentals + netkeiba odds dynamics, normalized into a replayable bronze→silver→gold lake.

INTEGRITY

Point-in-time features

Every feature respects what was knowable at race time — no leakage, validated with walk-forward backtesting.

MODELING

AI-assisted analysis

Within-race standardized indices and ML models that surface notable horses, with predictor scoring.

HONESTY

Published validation

Accuracy and recovery measured and shown openly. Curated results published here; raw data stays local.
